# AI Tools Analysis Project

## 📌 Project Overview

This project focuses on **analyzing AI tools using Data Science techniques in Python**.
The goal of this project is to explore datasets related to AI tools and perform **data cleaning, analysis, and visualization** using popular Python libraries.

The project demonstrates how data science can be used to understand trends, usage patterns, and insights from AI-related datasets.

---

## 🎯 Objectives

* Understand the dataset of AI tools
* Perform **data preprocessing and cleaning**
* Analyze important trends in AI tools
* Create meaningful **data visualizations**
* Extract insights using **Python data science libraries**

---

## 📊 Data Analysis Process

### 1️⃣ Data Collection

The dataset containing information about AI tools was collected and stored in the **data folder**.

### 2️⃣ Data Cleaning

Using **Pandas**, the dataset was cleaned by:

* Handling missing values
* Removing duplicates
* Formatting data columns

### 3️⃣ Data Exploration

Exploratory Data Analysis (EDA) was performed using:

* Summary statistics
* Data distribution analysis
* Feature relationships

### 4️⃣ Data Visualization

Visualization was done using:

* **Matplotlib**
* **Seaborn**

These helped in understanding:

* Popular AI tool categories
* Usage trends
* Data distributions

## 🚀 How to Run the Project

1. Clone the repository


2. Open the project folder

3. Install required libraries

```

```

4. Run the Jupyter Notebook

```
jupyter notebook
```
